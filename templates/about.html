About This Project

Welcome to the AI-Powered Plagiarism Checker, developed as a full-stack web application using Flask, SQLite, HTML, and CSS.

This platform allows users to upload and compare documents in multiple formats including PDF, DOCX, TXT, and even code files like Python, Java, and HTML. It tokenizes content, filters out comments, and compares with previously stored documents for potential similarities.

It was built with the intention to help students, educators, and professionals maintain originality and avoid plagiarism using smart comparison algorithms.

Designed and developed by Chandini Tadicharla as part of a passion project to apply AI and web development in real-world problem-solving.
